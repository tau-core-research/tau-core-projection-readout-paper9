# Data Notice

This public paper package contains manuscript sources, compact derived protocol
tables, and generated figures only.

It does not redistribute raw SPARC rotation-curve files, raw survey products,
velocity-field cubes, private workbench products, or broad Tau Core theory-hub
drafts. Paper 1, Paper 2, and Paper 3 are used as methodological and derived
artifact context, but this package does not copy their raw input data.

The package is a forward-gate and reproducibility-protocol paper. It defines a
morphology-matched readout test that can be run later on accepted component
tables and residual-blind morphology labels. It does not report an empirical
galaxy-fit victory.
